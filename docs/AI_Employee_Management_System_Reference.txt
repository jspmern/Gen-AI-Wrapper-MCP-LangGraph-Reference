HUMAN IN THE LOOP (HITL)

Sensitive Actions:
- deleteEmployee
- updateSalary
- approveReimbursement

Flow:
Agent -> Interrupt -> Manager Approval -> Resume -> Execute Tool

Example:
interrupt({
 action: "DELETE_EMPLOYEE",
 employeeId: "123"
})

After approval:
graph.resume("approved")
